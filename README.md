# Crunchyroll Better Web

This project builds on an extension originally made by [Conscious_Bell_8765](https://www.reddit.com/user/Conscious_Bell_8765/), published as [Crunchyroll Better Web on the Chrome Web Store](https://chromewebstore.google.com/detail/crunchyroll-better-web/fehhjmlpnknlielfademfbbpkicjlfdh?hl=en-US&utm_source=ext_sidebar).

The original extension had become outdated, especially around loading newer anime seasons correctly, and it was also missing a few homepage and player features that I wanted. This version keeps the core idea intact while updating the behavior and adding those missing pieces.

## Why This Version Exists

Crunchyroll's site structure changes over time, and the older version was no longer properly handling newer season data. I also wanted more control over the homepage layout and better keyboard behavior in the player, so this version extends the original work with those improvements.

## Features Added

- **Dashboard Layout Customization**  
  Added a homepage dashboard that lets you manage section visibility, ordering, and layout behavior more directly.

- **More Continue Watching**  
  Expanded the Continue Watching experience so more items can stay visible and easier to access.

- **Remove Crunchyroll Ads From Home Page**  
  Added an option to hide Crunchyroll promotional ad sections on the homepage for a cleaner layout.

- **Skip Shortcuts Without the Player Selector**  
  Enter, Space, and Left/Right Arrow support now works for skip actions without needing to focus or rely on the player selector first.

## Installation

### Load It Locally in Chrome

1. Download or clone this repository.
2. Open `chrome://extensions` in Chrome.
3. Turn on **Developer Mode**.
4. Click **Load Unpacked**.
5. Select the `1.8_0` folder.
6. Open Crunchyroll and use the extension popup to configure the features you want.

### Original Extension

If you want the original release, you can find it here: [Crunchyroll Better Web on the Chrome Web Store](https://chromewebstore.google.com/detail/crunchyroll-better-web/fehhjmlpnknlielfademfbbpkicjlfdh?hl=en-US&utm_source=ext_sidebar).
